# ✈️ TripGenius AI

An AI-powered travel planner that creates personalized travel itineraries based on:

- 📍 Destination
- 📅 Duration
- 💰 Budget
- 🎯 Interests

## 🚀 Features

- AI-generated travel itineraries
- Hotel recommendations
- Tourist attractions
- Local food suggestions
- Weather information
- Interactive map
- Budget planning

## 🛠️ Tech Stack

### Frontend
- React
- Vite

### Backend
- FastAPI
- Python

### AI
- Google Gemini API
